# Pet Planner v3 🐾

This version adds the first proper pet profile system.

## New features

- Add unlimited pets
- Choose animal type
- Save name, breed/species, birthday, sex and weight
- Switch between pet dashboards
- Animal-specific habitat builder names
- Delete pet profiles
- Data saves locally on the device

## Upload

Replace your existing `index.html` and `style.css`, then add `app.js`.

You can also replace `manifest.json` and `README.md` with the new versions.
