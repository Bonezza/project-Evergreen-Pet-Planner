# 🌿 Project Evergreen

## Version 8.1 — Polish & Builder Upgrade

Project Evergreen is a digital home for every animal that matters to someone: pets, wild friends, rescues, fosters and memorials.

## New in v8

- Welcome Home dashboard
- Today’s tasks and recent activity
- Favourite animal profiles
- Health status badges
- Animal category filters
- Gallery captions, favourites, viewer and delete controls
- Categorised notes
- Repeating reminders
- Planner completion tracking
- Wild Friend trust level
- Improved profile migration
- Export and import backups
- Habitat item selection, resize, rotate, duplicate and remove
- Separate saved layouts by animal, habitat type and size

## Upload to GitHub

Replace:

- `index.html`
- `style.css`
- `app.js`
- `manifest.json`
- `README.md`
- `VISION.md`

Existing profiles from previous versions should migrate automatically on the same browser/device.

## Important data note

The current web app stores data in browser localStorage. Data does not automatically sync between devices. Use **Settings → Export backup** before moving devices or clearing browser data.
